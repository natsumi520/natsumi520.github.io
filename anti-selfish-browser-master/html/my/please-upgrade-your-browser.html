သင်၏ browser ကိုအဆင့်မြှင့်တင်ရန်အချိန်တန်ပြီ။

သင်သည် Internet Explorer ဟောင်းကိုအသုံးပြုနေသည် (သို့မဟုတ် kernel browser ကို သုံး. ) ။ဆိုလိုသည်မှာသင်သည် browser ကိုအဆင့်မြှင့်တင်ခြင်းမပြုမီဤဝက်ဘ်ဆိုက်ကိုသင်ဝင်ရောက်နိုင်မည်မဟုတ်ပါ။

မှတ်ချက် - Internet Explorer ၏ပံ့ပိုးမှုဝန်ဆောင်မှုကိုရပ်ဆိုင်းလိုက်သည်

2016 ခုနှစ်, ဇန်နဝါရီလ 12 ရက်နေ့မှစ. Microsoft သည်အောက်ဖော်ပြပါ IE 11 အတွက်သက်ဆိုင်ရာထောက်ခံမှုနှင့်အသစ်ပြောင်းခြင်းများကိုမပေးတော့ပါ။အဓိက browser လုံခြုံရေးမွမ်းမံမှုမရှိပါ, သင်၏ကွန်ပျူတာသည်သင်၏စီးပွားရေးအချက်အလက်နှင့်သတင်းအချက်အလက်များကိုခိုးယူနိုင်သည့်အန္တရာယ်ရှိသောဗိုင်းရပ်စ်များ, spyware နှင့်အခြား malware တိုက်ခိုက်မှုများကိုအားနည်းစေနိုင်သည်။မြင်Internet Explorer အဟောင်းအတွက် Microsoft ၏ပံ့ပိုးမှုဝန်ဆောင်မှုကိုရပ်ဆိုင်းလိုက်သည်။

2020 ခုနှစ်နိုဝင်ဘာလ 30 ရက်မှစ. Microsoft အသင်းများကို 0 င်ရောက်ရန် Internet Explorer 11 ကိုကျွန်ုပ်မသုံးနိုင်ပါ။ကျန်ရှိသော Microsoft 365 application သည် 2021 ခုနှစ်သြဂုတ်လ 17 ရက်နေ့တွင် IE11 ကိုမထောက်ပံ့တော့ပါ။ဘရောက်ဇာကိုယ်တိုင်က Windows ဗားရှင်း၏ Windows ဗားရှင်း၏ 0 င်းဒိုးဗားရှင်းကို 2022 ခုနှစ်, ဇွန်လ 15 ရက်အထိဘ 0 တစ်ခုလုံးကိုကျော်ဖြတ်နိုင်သည်။မိုက်ကရိုဆော့ဖ်သည် Internet Explorer အသုံးပြုသူများကို EDGE သို့ပြောင်းရွှေ့ရန်နှင့်အင်တာနက်အပလီကေးရှင်းဟောင်းများကိုထောက်ပံ့သော "Internet Explorer Mode" ကိုအသုံးပြုရန်အကြံပြုသည်။

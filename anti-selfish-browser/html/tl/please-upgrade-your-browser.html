Gumagamit ka ng isang lumang bersyon ng Internet Explorer (IE11 o sa ibaba o isang browser gamit ang kernel na ito).Nangangahulugan ito na hindi mo ma-access ang website na ito hanggang sa i-upgrade mo ang iyong browser.

Mangyaring tandaan: Ang mga serbisyo ng suporta para sa Internet Explorer ay hindi na ipinagpatuloy

Pagkatapos ng Enero 12, 2016, hindi na nagbibigay ang Microsoft ng kaukulang suporta at mga update para sa IE 11 at sa ibaba.Nang walang kritikal na mga update sa seguridad ng browser, ang iyong computer ay maaaring mahina sa mga nakakapinsalang virus, spyware, at iba pang malisyosong software na maaaring magnakaw o makapinsala sa iyong data ng negosyo at impormasyon.

Mula noong Nobyembre 30, 2020, ang web version ng mga koponan ng Microsoft ay hindi na ma-access gamit ang Internet Explorer 11. Ang natitirang mga aplikasyon ng Microsoft 365 ay magbabawas ng suporta para sa IE11 sa Agosto 17, 2021. Ang browser mismo ay patuloy na suportado para sa lifecycle ngAng bersyon ng Windows kung saan ito ay naka-install hanggang Hunyo 15, 2022.microsoft inirerekomenda ng mga gumagamit ng Internet Explorer lumipat sa gilid at gamitin ang built-in na "Internet Explorer Mode" na nagbibigay-daan sa suporta para sa mga application ng Legacy Internet.
